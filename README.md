# KilledByDiscord

This repo is where `killedbydiscord.rip` fetches data from.

### Want to add a product?
Edit `products.json`, following the format below:
```json
{
  "name": "Product name",
  "description": "Product name was...",
  "created": "XXXX",
  "killed": "XXXX",
  "icon": "URL - optional"
}
```
